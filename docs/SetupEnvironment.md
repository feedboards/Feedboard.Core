# install dependencies

```bash
yarn
# or
npm i
```

# `Build` and `Watch` the library

```bash
# build
yarn build
# or
npm run build

# watch
yarn watch
# or
npm run watch
```

# Test in console

Create in root folder `test.cjs` file then import

```js
import { ... } from './dist/index.js'

// do something
```

run

```bash
node ./test.cjs
```
